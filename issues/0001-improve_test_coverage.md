# 0001 - Improve Test Coverage

## 问题描述
当前项目的测试覆盖率较低，部分代码未被测试覆盖。为了提高代码质量和可靠性，需要增加测试覆盖率。

## 解决方案
1. 更新 `pytest.ini` 文件，添加覆盖率插件配置：
   ```ini
   addopts = --cov=qlib --cov-report=term-missing --cov-report=html
   ```
2. 运行测试并生成覆盖率报告。

## 结果
- 覆盖率配置已成功添加到 `pytest.ini`。
- 测试运行成功，覆盖率数据已收集。

## 后续建议
- 编写更多测试用例以覆盖未被测试的代码部分。
- 定期检查覆盖率报告，确保新代码被充分测试。
